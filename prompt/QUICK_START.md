# 🚀 快速开始

## 3分钟上手

### 1️⃣ 安装依赖
```bash
pip install -r requirements.txt
```

### 2️⃣ 查看可用策略
```bash
python src/cli.py strategies
```

### 3️⃣ 分析你的项目
```bash
python src/cli.py analyze .
```

### 4️⃣ 应用优化
```bash
python src/cli.py optimize .
```

## 🎯 常用命令

| 命令 | 功能 | 说明 |
|------|------|------|
| `strategies` | 查看优化策略 | 列出所有可用策略 |
| `analyze .` | 分析项目 | 扫描代码问题 |
| `optimize .` | 优化代码 | 应用安全策略 |
| `self-opt .` | 自优化模式 | AI自我改进演示 |
| `demo` | 快速演示 | 展示完整功能 |

## 🌟 立即体验

```bash
# 5分钟完整体验
python src/cli.py demo && echo "✨ 完美！"
```
