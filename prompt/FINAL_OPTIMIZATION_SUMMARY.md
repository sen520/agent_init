```bash

# 2. 查看可用策略

# 3. 分析当前项目
python src/cli.py analyze .

# 4. 应用优化
python src/cli.py optimize .

# 5. 体验自优化
python src/cli.py self-opt .
```

### 🎛️ **高级用法**
```python
# 编程方式使用

# 使用配置

optimizer = CodeOptimizer()

# 应用优化
result = optimizer.optimize_file("your_code.py", strategies)
```
